# 📊 YaAd Dashboard — Статистика мобильной рекламы

### AdStat — это мини-приложение для iOS, которое помогает разработчикам и издателям мобильных приложений отслеживать статистику заработка на рекламе. Просто введите API-токен, выберите сеть и период — и получите актуальные данные о доходах.

## Возможности

### 📱 Поддержка двух рекламных сетей:
* Рекламная сеть Яндекс (РСЯ)
* Простая монетизация (Медиация)

### 📅 Гибкий выбор периода:
* Сегодня
* Вчера
* Последние 7 дней
* Последние 14 дней
* Последние 30 дней
* Последние 90 дней

### 💰 Выбор валюты:
* Российский рубль (₽)
* Доллар США ($)

---
Свой токен можно получить в [личном кабинете РСЯ](https://partner.yandex.ru)
